# learnqml
A qml course for complete beginners

## Contribute
To contribute, you'll need to install mdbook.
Follow the instructions at the [rust website](https://rust-lang.github.io/mdBook/guide/installation.html).

To preview the changes made the course directly in your browser, you may use ```mdbook serve --open``` at the root of the project.

[Link to the website](https://nicolasgourdiersurgar.github.io/learnqml/)

## Admonish
We use the Admonish plugin to add stylised blocks. You can find a list of available directives [here](https://tommilligan.github.io/mdbook-admonish/reference.html#directives)
